These are projects that I have embarked on. I'm still working on some, have abandoned a few, but none are complete. All of these projects are open source.

  • Git for Beginners Workshop

    Part of hackerschool, a series of workshops NUS Hackers conduct every semester. I have conducted this twice.

  • HgVis

    A visualization tool for Mercurial commits, with the ability to view commits per dev per day.

  • QuickView for YouTube

    Chrome plugin to allow you to watch videos by hovering over the thumbnail.

  • any2048

    2048 clone with a twist, lets you add any pictures to replace numbers

  • Snowball

    HTML5 + JavaScript game that me and my partner made as part of a summer independent module. Control and snowball rolling down the hill and avoid obstacles!

  • NUSMorge

    Timetable merger so you can visualize when schedules clash. Won Best Freshmen Project at Hack n Roll 13.

  • Dose

    Android app that reminds you to take medicine

  • CodeMirror

    In-browser code editor. I contributed to the code base.

  • Ganglia

    Distributed monitoring system, contributed to the upstream libraries.
